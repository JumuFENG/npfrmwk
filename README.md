npfrmwk
=======

a framework for npapi plugin developing 
简要说明：
项目依赖cmake。如需编译请自行下载cmake安装，然后如果是VS2010，直接运行cmake(VS2010).bat。如果是其他编译器请参考CMAKE中使用相关编译器生成项目的参数。
你自己可以在两个例子的基础上修改以开发npapi插件，只需要修改rc文件中相应的信息即可。
也可以将示例项目复制一份，更改文件夹名称，更改rc文件信息之后，修改CMakeLists.txt文件然后生成新的项目。CMakeLists.txt文件的修改可以参考已有文件也可以自行按照cmake的规范编写。
如果对该项目感兴趣，愿意在该框架的基础上贡献更多的示例，可以联系作者，共同维护该项目。
